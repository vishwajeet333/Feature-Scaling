# Feature Scaling & Engineering

This repository explores essential techniques for preprocessing data to improve model performance.

**Topics:**
* **Standardization & Normalization:** Scaling features using StandardScaler and MinMaxScaler.
* **Categorical Data Handling:** Implementing OrdinalEncoding and OneHotEncoding.
* **Advanced Transformations:** Using PowerTransformer and FunctionTransformer for distribution adjustment.
* **Pipeline & ColumnTransformer:** streamlined preprocessing workflows.
